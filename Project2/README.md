# Fleet Management System

This Java-based Fleet Management System allows users to manage a fleet of boats, track expenses, and add new data between runs.

## Features
- Load fleet data from a CSV file (initial run)
- Save and load fleet data using serialized .db file (subsequent runs)
- Add, remove, and update boat expenses
- Print fleet reports
